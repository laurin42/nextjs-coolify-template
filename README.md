# Next.js Starter Template

## Key Features

- **Framework:** Next.js (App Router) with TypeScript
- **Hosting:** Optimized for **Coolify** (Nixpacks & Docker standalone support)
- **Animation:** Integrated **GSAP** with **ScrollSmoother** for premium UX
- **Navigation:** Dynamic **Active Section Detection** for the Header
- **Styling:** Tailwind CSS with a modular component structure
- **Pages:** Pre-configured routes
- **Layout:** Responsive Header/Footer with a seamless content wrapper

## Tech Stack

- Next.js 16
- GSAP & ScrollSmoother
- Tailwind CSS
